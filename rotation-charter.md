# Rotation Charter

The spine of the system. One head pursuit and one hands pursuit per season; everything else parked. Reviewed quarterly. Fitness and dog walks sit outside the rotation — bedrock, not contenders.

**Builder's rules:** resource docs exist only for active pursuits. The parked list stays a plain list. If the system itself becomes the hobby, stop tinkering and decline *mensa*.

---

## Current season: Summer 2026 (June–August)

**Head — Latin.** Romano-British angle: case system first, building toward reading real inscriptions. Resource library: the three HTML reference volumes (cases & prepositions; all five declensions; adjective agreement). Duolingo continues for vocab only.

**Hands — Knots.** Pocket cord, telly-compatible. Target: a working repertoire (bowline, clove hitch, taut-line, sheet bend, trucker's hitch, figure-eight family) reliable enough to tie without thinking.

**Substrate (always on) — Writer's notebook.** Ideas, fragments, images. No drafting obligation. Feeds future fiction season.

**Opportunistic — Greenwood carving.** Sunday afternoons when materials allow. No guilt when they don't.
